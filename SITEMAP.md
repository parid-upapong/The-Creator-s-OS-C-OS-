# Sitemap: Creator OS Web Application

## 1. Authentication & Onboarding
- /login (Social Login: Google, Apple)
- /onboarding (Niche selection, Platform connection)

## 2. Dashboard (The Command Center)
- /dashboard (Overview of recent projects & performance)
- /upload (Central hub for importing Hero Content)

## 3. Project Workspace
- /project/:id/overview (Manage all assets derived from one source)
- /project/:id/video-editor (Vertical clip refinement)
- /project/:id/copy-editor (LinkedIn/X/Blog post refinement)
- /project/:id/localization (Translation & Dubbing settings)

## 4. Content Library
- /library/assets (Storage for raw and processed media)
- /library/templates (Save custom brand styles, fonts, and colors)

## 5. Scheduler & Distribution
- /calendar (Visual view of upcoming posts across all platforms)
- /integrations (Manage API keys for TikTok, YT, IG, etc.)

## 6. Analytics
- /analytics/cross-platform (Comparative view: which platform likes this content more?)
- /analytics/roi (Time saved vs. engagement growth)

## 7. Account & Billing
- /settings/profile
- /settings/billing (Subscription tiers based on AI minutes/usage)
- /settings/team (Multi-user access for agencies)
